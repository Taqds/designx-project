# DesignX Agency Portfolio Website

This is a premium, high-performance agency website for **DesignX**, a top-rated web design agency based in Fort Lauderdale, Florida.

## 🚀 Live Demo
The project is live at: [https://design-x-web.netlify.app/](https://design-x-web.netlify.app/)

## ✨ Key Features
- **High-Converting Design**: Custom-built with modern UI/UX principles.
- **Fully Responsive**: Optimized for all screen sizes (Mobile, Tablet, Desktop).
- **SEO Optimized**: Includes Sitemap.xml, Canonical tags, and Meta tags for better Google rankings.
- **Google Analytics Integrated**: Ready to track visitor data and conversion metrics.
- **Performance Focused**: Blazing fast load times with clean code.

## 🛠️ Project Structure
- `index.html`: Homepage with full agency services.
- `services.html`: Detailed breakdown of our offerings.
- `process.html`: Our specific workflow and strategy.
- `results.html`: Case studies and proven results.
- `reviews.html`: Client testimonials and social proof.
- `blog.html` & `blog-single.html`: Content hub for SEO and insights.
- `faq.html`: Frequently asked questions.
- `sitemap.xml`: Auto-generated sitemap for search engines.

## 📦 Deployment
This site is connected to **Netlify** via GitHub. Any push to the `main` branch will trigger an automatic deployment.
